# Login & Signup Page UI

A clean, modern, and fully responsive Login and Signup page user interface. Built as a beginner web development project using only HTML and CSS, with no JavaScript or backend. This is a design/UI-only project.

## Features

- Login card (email, password)
- Signup card (name, email, password, confirm password)
- Gradient background over an Unsplash image
- Focus styles on input fields
- Styled buttons with hover effects
- Fully responsive (cards sit side by side on desktop, stack on mobile)

## What I Learned

- How to build clean **form layouts** with labels and inputs
- How to **center content** using Flexbox
- How to add a **gradient over a background image**
- How to create **focus styles** and **hover effects**
- How to make a layout **responsive** with media queries
